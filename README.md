# HouseList Flutter App

## Project Description
HouseList is a Flutter application designed to help users manage and organize their house listings. Whether you are a real estate agent or a homeowner, this app serves as a great tool to streamline the process of listing and viewing houses.

## Features
- User-friendly interface
- Easy listing management
- Search and filter options
- Ciepły design color palette 
- Built with Flutter for cross-platform support

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/thclotheshop-pixel/HouseList.git
   ```
2. Navigate to the project directory:
   ```bash
   cd HouseList
   ```
3. Install the required dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
